# Logo

Логотип.

# Использование блока

``` js
{
    block : 'logo'
}
```

## Модификаторы блока

### _theme

- логотип ОАО «Альфа-Банк» (alfa)

``` js
{
    block : 'logo',
    mods : { theme  : 'alfa-on-color' }
}
```

- для белого фона (white)

``` js
{
    block : 'logo',
    mods : { theme  : 'alfa-on-white' }
}
```

- для цветного фона (color)

``` js
{
    block : 'logo',
    mods : { theme  : 'alfa-on-color' }
}
```

### _size

- маленький (s)

``` js
{
    block : 'logo',
    mods : { size : 's' }
}
```

- средний (m)

``` js
{
    block : 'logo',
    mods : { size : 'm' }
}
```

- большой (l)

``` js
{
    block : 'logo',
    mods : { size : 'l' }
}
```

- самый большой (xl)

``` js
{
    block : 'logo',
    mods : { size : 'xl' }
}
```

### _view

- логотип с текстом (full)

``` js
{
    block : 'logo',
    mods : { view : 'full' }
}
```

- только логотип (icon)

``` js
{
    block : 'logo',
    mods : { view : 'icon' }
}
```
