Collapse
========

Блок, содержимое которого можно скрыть/показать по запросу пользователя.

```js
    {
        block : 'collapse',
        mods : { theme : 'alfa-on-color' },
        expandedLabel : 'Скрыть подробности',
        collapsedLabel : 'Подробнее',
        content : [ 'Подробности...' ]
    }
```
