#menu

## Модификаторы блока

### Темы `_theme`

Блоку добавлена тема:
    * alfa-on-color
    * alfa-on-white

Наглядно показано на примере ниже:

```bemjson
 {
    block : 'menu',
    mods : { theme : 'alfa-on-color' }
 }
```

### Размер `_size`

- маленький (s)

```bemjson
{
    block : 'menu',
    mods : { size : 's' }
}
```

- средний (m)

```bemjson
{
    block : 'menu',
    mods : { size : 'm' }
}
```

- большой (l)

```bemjson
{
    block : 'menu',
    mods : { size : 'l' }
}
```

- самый большой (xl)

```bemjson
{
    block : 'menu',
    mods : { size : 'xl' }
}
```
